# Österreich
Eine eigene Foodosoft Instanz am [IG Foodcoops](https://foodcoops.at/)-Server die unter https://app.foodcoops.at/NAME-EURER-FOODCOOP erreichbar ist, könnt ihr per E-mail an support@foodcoops.at anfordern. 
{.is-danger}
